过滤器

过滤器是一种只能使用在表达式中，对表达式结果进行过滤的对象

注册 

//注册过滤器
impex.filter('lower',{
	to:function(){
        return this.$value.toLowerCase();
    }
})

<!-- 使用过滤器 -->
<div>
	{{ ' HELLO IMPEX !' => lower}}
</div>
{{ ' HELLO IMPEX !' => lower}}

注册过滤器不需要前缀，使用时也不需要前缀

过滤器只能是用在表达式中，文本表达式或者属性表达式都可以

参数和过滤链

过滤器可以有参数，这样可以用不同的过滤方案来过滤不同的表达式， 同时可以连接多个过滤器进行过滤， 这里可以看到具体应用的demo
