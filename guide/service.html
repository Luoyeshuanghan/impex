服务

服务是一种为其他组件、指令、过滤器、服务提供功能的对象，服务没有视图， 也不能在视图上使用

注册 

//注册服务
impex.service('HTTP',{
	get : function(url,cbk){
		var xhr;
		//...
	}
});
},['HTTP'])

服务中的内容可以任意定义，可以弹出框也可以请求网络也可以打印日志，只要记住，这些函数 最终都会被其他impex对象内部调用

注入 


//匿名组件
impex.render(document.body,{
	onCreate:function(http){
		http.get(url,...)
	}
},['HTTP']);

//命名组件
impex.component('x-iframe',{
	onCreate:function(http){
		http.get(this.url,function(content){
			this.content = content;
		})
	}
},['HTTP']);

//过滤器
impex.filter('date',{
	onCreate:function(df){
		this.df = df;
	},
	to:function(){
		return this.df.format(this.$value);
	}
},['DateFormat'])

服务可以被注入到组件、指令、过滤器或者另一个服务中，所有注入的服务都会作为 onCreate的参数，按照注入顺序

模式

默认情况，当服务要被注入时都会创建一个该服务的实例，但如果你需要的是一个单例服务，比如纯函数服务。 impex.js 允许在注册服务时，使用$singleton属性声明单例模式 

//注册服务
impex.service('HTTP',{
	$singleton : true,//singleton mode
	get : function(url,cbk){
		var xhr;
		//...
	}
});
},['HTTP'])
